Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas nao considerado. Utilisando a tag DEL. (semântica)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Utilizando a tag INS. (semântica)

Existe também o sublinhado com a tag U (não semântica)

Texto subscrito

para inserir coisas do tipo H2O. Utilizamos a tag SUB
